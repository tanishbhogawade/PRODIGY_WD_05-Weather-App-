# PRODIGY_WD_05

## Weather Web Application

### 📌 Description
A weather application that fetches real-time weather data using an API and displays it dynamically to the user.

### ✨ Features
- Fetch weather data using API
- Display temperature and conditions
- Location-based search
- Dynamic UI updates

### 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

### 🎯 Learning Outcomes
- API integration
- Fetching data using JavaScript
- Dynamic DOM updates
